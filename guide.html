Everything your school needs to know about SafeTrack — how it works, how staff use it, how parents use it, and how your school earns from it.

🛡️ What is SafeTrack?

SafeTrack is a school safety system that automatically notifies parents the moment their child arrives or leaves school. Every student gets a unique QR code on their ID card. When staff scan the card at the gate, parents instantly receive a WhatsApp message with the exact time.

SafeTrack also includes a secure PIN system for child pickup — ensuring no child is released to an unauthorized person.

👨‍💼 How School Staff Use SafeTrack

1

Open the Gate Scanner

Every morning, the gate staff opens SafeTrack on their phone or tablet by going to the school's SafeTrack link. No app download needed — it works directly in the browser.

2

Select Arrival or Departure

In the morning, tap the green "Arrival" button. In the afternoon when students are leaving, tap the red "Departure" button.

3

Scan the Student's ID Card

Point the phone camera at the QR code on the student's ID card. The system beeps and shows the student's name and time. The parent is automatically notified via WhatsApp.

4

Verify Pickup PIN

When someone comes to pick up a child, switch to the "Verify PIN" tab. Ask them for their 6-digit PIN. Enter it into the system — it will show the child's name and the authorized pickup person's name. Ask the person to confirm their name before releasing the child.

⚠️ Important Security Rule

Always ask the pickup person to say their name BEFORE showing them the screen. The name on screen must match what they say. If it does not match — do NOT release the child and contact the parent immediately.

👨‍👩‍👧 How Parents Use SafeTrack

📱

Install the App

Parents open the SafeTrack parent portal in their phone browser and add it to their home screen like a real app. Works on Android and iPhone.

🔑

Login with QR Code

Parents login using their phone number and their child's QR code — found in the ID card booklet given at registration.

📊

View Attendance

Parents can see their child's full attendance history — every arrival and departure with exact date and time.

🔐

Generate Pickup PIN

When sending someone to pick up their child, parents generate a secure 6-digit PIN valid for 30 minutes and share it with the pickup person.

🖥️ How School Admin Uses SafeTrack

1

Login to School Dashboard

The school admin logs in with their unique username and password provided by SafeTrack during onboarding.

2

Register Students

Add each student to the system with their full name, class, parent phone number and WhatsApp number. The system automatically generates a unique QR code for each student.

3

View Attendance Reports

The dashboard shows today's arrivals and departures in real time. Use the date filter to view attendance for any previous date.

🔒 Three Layer Security System

SafeTrack is the only school safety system in Nigeria with three independent security layers protecting every child.

1

QR Code — Unique to every student

Each student has a unique QR code that cannot be duplicated. Lost cards can be reported and deactivated immediately.

2

Time-Limited PIN — Expires in 30 minutes

Parents generate a new PIN every time someone needs to pick up their child. The PIN expires after 30 minutes and can only be used once.

3

Name Verification — Human security check

Staff ask the pickup person to state their name before checking the system. If the name does not match — the child is not released.
